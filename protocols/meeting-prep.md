# 会議準備・議事録

会議の前後をサポートする。「準備して」→ 事前準備、「議事録」→ 記録作成。

## 事前準備（会議前）

### 実行手順

1. カレンダーから対象会議の情報を取得（参加者・議題・資料リンク）
2. 過去の関連議事録があれば `areas/meeting-notes/` から検索
3. 準備メモを生成

### 出力フォーマット

```
## [会議名]（YYYY-MM-DD HH:MM）

### 参加者
- [名前（役割）]

### 議題
- [議題1]
- [議題2]

### 前回からの持ち越し
- [前回の議事録から未完了アクションを抽出]

### 準備事項
- [必要な資料・データ・確認事項]
```

## 議事録（会議後）

ユーザーが会議内容をテキスト/音声書き起こしで共有した場合に作成する。

### 出力フォーマット

```
## 議事録: [会議名]（YYYY-MM-DD）

### 参加者
- [名前]

### 決定事項
- [決まったこと]

### アクションアイテム
| 担当 | タスク | 期限 |
|------|--------|------|
| [名前] | [内容] | [日付] |

### 議論メモ
- [重要な議論のポイントを簡潔に]

### 次回予定
- [次回の日程・議題]
```

## 保存

`areas/meeting-notes/YYYY-MM-DD-[会議名].md` に保存する。
アクションアイテムに自分のタスクがあれば、projects/ への登録を提案する。
