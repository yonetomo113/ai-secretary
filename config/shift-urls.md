# シフト表URL

毎月URLが変わるため、翌月分が届いたら **新しいセクションを追記** する。
古い月のセクションは削除せず残す（履歴として保持）。

## 更新ルール
- 毎月24日17:00に assift_automator.py が自動実行される
- 実行前に assift で翌月分URLを発行し、以下フォーマットで追記すること
- ヘッダーを `## YYYY年M月` 形式にすることで automator が正しい月のURLを選択する

追記フォーマット:
```
## 2026年5月

| 物件 | URL |
|------|-----|
| 竹屋旅籠 | https://assift.com/share/XXXXXXXX |
| 登竜庵   | https://assift.com/share/YYYYYYYY |
```
