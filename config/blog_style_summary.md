## 2. 文章の長さ・リズムの特徴

### 一文の長さ
- **基本は短文〜中文の混在型**
- 短文（10〜20字）と長文（50〜100字超）を意図せず交互に配置する傾向
- 「さて当日。」「朝から身支度整え西蟹屋へ。」のような**極短文でシーン転換**するのが特徴的
- 長い一文は読点を挟まず息継ぎなしで続けることも多い（口語的な「勢い」重視）

### 句読点の使い方
- 読点（、）は**少なめ**。口語の「流れ」を優先し、文法的な正確さより読んだときの自然さを重視
- 「。。。」（三点リーダー代わりの句点連打）が多用される→**余韻・迷い・脱力感**の表現
- 感嘆符「！」は感情の高まりに使い、絵文字と連続させることが多い
- 文末に`ｗ`をつけてから句点を省略するケースが多い

### リズムの型
```
【短文で状況提示】→【長めの説明・経緯】→【短文オチ or ｗで締め】
```

## 3. 段落構成のパターン

### 基本型：「現場実況レポート型」

```
① 導入：近況・日付・場所のさらっとした提示
② 経緯説明：「〜だったのですが」でつなぐ背景説明
③ 現場描写：箇条書き的な短文の連続でテンポよく進める
④ トラブル・誤算の開示：「思いの外〜」「まさかの〜」で意外性を出す
⑤ 対処・結果：さっぱりと報告
⑥ 感想・オチ：ｗ・絵文字・自虐で締め
⑦ 次回予告 or 宣伝（任意）：「次回〜」「来広の際はぜひ」
```

### サブパターン：「自問自答型」
- 問い（「どうしようか？」）→迷い（「うーん」）→決断（「と言うわけで」）→実行報告
- 内部の思考プロセスをそのまま書き起こしたような構成

### 特記：時系列が前後する
- 「時を前後しますが」「順番前後しますが」と**明示した上で**過去に戻る
- 記憶の順番通りに書く＝日記的な自然体の構成

## 4. よく使うカテゴリ・テーマと扱い方

| カテゴリ | 扱い方の特徴 |
|---|---|
| **物件改装・DIY記録** | 作業工程を現場実況風に。失敗も隠さず「アホの極み」と自虐込みで報告 |
| **旅館業・民泊運営** | 許認可の苦労、行政への不満を率直に。「いやそんなの知らんがな」と本音をそのまま出す |
| **トラブル・修繕** | 「お金が飛んでいく」系エピソードを自嘲まじりに記録。深刻になりすぎず笑いに変換 |
| **飲食・旅行** | 食レポは短めかつ感覚的。「うんまい！」「美味しゅうございました」が定番 |
| **不動産投資の数字検証** | このカテゴリのみ文体がやや硬くなり、数値を丁寧に並べる。他記事との温度差あり |
| **人間関係・仲間紹介** | 登場人物を「よしゆきさん」「mayaさん」と実名に近い形で親しみを込めて登場させる |

## 5. 読者への語りかけ方

### 一人称
- **「僕」** で統一（「私」はほぼ使わない。新年挨拶など公式寄り記事のみ「私」が出現）

### 読者への呼びかけ
- 直接的な呼びかけは少なく、**読者を横に置いた独り言スタイル**が基本
- 「〜見てもらうとして」「〜聞いてください」など**軽く視線を向ける程度**
- 「皆様」は宣伝・告知パートのみ登場するフォーマル語
- 「来広の際はぜひご利用下さいませ」が定番の締め文句（宣伝時）

### 読者との距離感
- **友人に話すような距離感**。上から教えるでも媚びるでもない
- コメントや反応を想定した「ｗ」「知らんけど」が読者を意識した装置として機能

## 6. 特徴的な語彙・専門用語の使い方

### 不動産・建築用語
- 専門用語（「汚水枡」「ピンホール」「プライマー」「ガンタッカー」「長尺シート」「特定小規模施設用自動火災報知設備」）を**説明なしでそのまま使う**
- 読者に一定の知識を前提として話す→**同業者・大家仲間が主要読者**と想定した書き方

### カタカナ・略語
- 「リノベ」「スイッチボット」「ステージング」「インダストリアル」「ガーリー」など
- 業界の日常語として自然に混ぜる

### 数字の出し方
- 金額はざっくり「7万円也」「47万円。。。吐き気がするｗ」のように**感情とセット**で出す
- 面積・距離も「広島駅徒歩10分」「11.9mを4分割」など具体数字をよく入れる

## 7. 感情表現・テンションの傾向

### テンションの振れ幅が大きい
- **ハイテンション**：「いやーーーーーーっ！」「かああああかっこよっ😤」「やあああああっと！」
  →母音を伸ばす・記号で連打・絵文字で表現
- **低テンション**：「。。。」「🥺」「激萎え」「ガクブル状態」
  →脱力系絵文字と三点リーダーで表現

### 怒り・不満の出し方
- 行政・役所への不満は**直接的**だが暴言にはならない
- 「いい加減にしろ」「アホかいな」程度の口語的なぼやき
- 「静かに切れた」など**抑えた表現で実は怒っている**ことを示す

### よく使う絵文字と意味
| 絵文字 | 使われる文脈 |
|---|---|
| 😤 | 気合・決意・やってやる感 |
| 🥺 | 困惑・お金が飛ぶ・泣き寝入り |
| ｗ / ｗｗｗ | 自嘲・ツッコミ・照れ隠し |
| 😘 | 冗談・おどけ・照れ |
| 🤔 | 思案中・疑問提起 |
| 😭 | 大げさな嘆き（本気ではない） |
